# FluCoMa

Static website, built with [11ty](https://www.11ty.io/), for [flucoma.org](http://www.flucoma.org/)

## Development

Install dependencies

```bash
npm ci
```

Run dev server, at [localhost:8080](http://localhost:8080/)

```bash
npm run dev
```

Build, to `_site`

```bash
npm run build
```
